# גווני צבע מהלוגו "פיג"ה" עבור אייקוני Lucide

פלטת צבעים שחולצה מהלוגו, לשימוש עקבי במערכת עם אייקוני Lucide Icons.

---

## פלטת צבעים (HEX)

| שם צבע | קוד HEX | שימוש בלוגו |
|--------|---------|-------------|
| **כחול ראשי** | `#2E499B` | כותרת ראשית, גוף הרקטה, מסגרת, שורת תחתית |
| **כחול משני** | `#4663AC` | טקסט "מעלה אדומים" |
| **כחול בהיר** | `#6896F0` | כוכבים/ניצוצות |
| **אדום הדגשה** | `#E22830` | סנפירי הרקטה |
| **כתום הדגשה** | `#FF8000` | להבת רקטה חיצונית, קשת טיסה כהה |
| **צהוב הדגשה** | `#FFCB00` | להבת רקטה פנימית |
| **כתום בהיר** | `#FFC07F` | קשת טיסה בהירה |
| **לבן** | `#FFFFFF` | רקע, חלון הרקטה |

## הנחיות עיצוב לאייקוני Lucide

### צבע אייקונים ראשי
- **כחול ראשי (`#2E499B`)** – צבע ברירת המחדל לרוב האייקונים
- משקף את הזהות המרכזית של המותג
- שימוש: ניווט, תפריטים, כפתורים ראשיים, אלמנטים מרכזיים

### אייקונים משניים
- **כחול משני (`#4663AC`)** – אייקונים פחות דומיננטיים
- שימוש: טקסט משני, תוויות, אלמנטים עזר

### הדגשה ואקשן
- **אדום הדגשה (`#E22830`)** – סכנה, שגיאה, מחיקה, אזהרות חמורות
- **כתום הדגשה (`#FF8000`)** – התראה, מצב פעיל, אלמנטים חדשים
- **צהוב הדגשה (`#FFCB00`)** – הודעות, הדגשת פריטים חשובים
- **כתום בהיר (`#FFC07F`)** – וריאציה עדינה להדגשה

### אלמנטים מיוחדים
- **כחול בהיר (`#6896F0`)** – פריטים חדשים, מועדפים, אלמנטים זוהרים

### רוחב קו (Stroke Width)
- מומלץ: `1.5`–`2` לשמירה על קריאות וחוזק ויזואלי
- בהתאם לסגנון הברור של הלוגו

### עקביות
- שמור על שימוש אחיד בצבעים לאורך המערכת
- חוויית משתמש מגובשת התואמת את מיתוג "פיג"ה"

## משתני CSS מומלצים

```css
:root {
  /* צבעי אייקונים */
  --icon-primary: #2E499B;
  --icon-secondary: #4663AC;
  --icon-accent: #6896F0;
  --icon-danger: #E22830;
  --icon-warning: #FF8000;
  --icon-highlight: #FFCB00;
  --icon-warning-light: #FFC07F;
}
```

## דוגמת שימוש (React + Lucide)

```tsx
import { Home, Settings, AlertTriangle, Star } from 'lucide-react';

// אייקון ראשי
<Home size={24} color="#2E499B" strokeWidth={1.5} />

// אייקון משני
<Settings size={20} color="#4663AC" strokeWidth={1.5} />

// אייקון אזהרה
<AlertTriangle size={24} color="#E22830" strokeWidth={2} />

// אייקון מועדף
<Star size={20} color="#6896F0" strokeWidth={1.5} fill="#6896F0" />
```
